## Site Navigation Element

Create a site navigation element.

[Schema.org SiteNavigationElement documentation](https://schema.org/SiteNavigationElement)

```markdown
---
elementType: SiteNavigationElement
identifier: navbar
inLanguage: en
itemListElement:
  - name: Home
    path: /
  - name: Website Section
    path: /en/section/
---
```
